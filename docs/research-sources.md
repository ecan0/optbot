Use this page to record academic sources, adapted instruments, and third-party creative materials. Add each source before using its ideas, wording, data, or assets in the study.

## Keep research data outside the repository

Participant responses, exports, identifiers, consent records, and derived private datasets don't belong in this public repository. The MIT License doesn't apply to research data that isn't included here.

Follow [Protect the public repository boundary](public-repo-boundary.md) before adding study or deployment artifacts.

## Review sources before submission

Before submitting the class project:

1. Match every in-text citation to the course bibliography
2. Confirm reuse rights for copied or adapted wording
3. Record the license and attribution for every third-party asset
4. Describe changes made to licensed materials
5. Confirm that no participant data or credentials are tracked
6. Ask the instructor about any source whose reuse terms remain unclear
